Channel → Section Rules

New channels matching a prefix get moved into the named sidebar section.

Auto-Read Prefixes

All messages in these channels get marked as read every minute — clears them from the Activity feed.

Auto-Read Invitations to:

When you're invited to a channel matching one of these prefixes, the invitation is auto-cleared. Real messages and @mentions in the channel stay unread.

Auto-Read Broadcasts in:

@channel / @here, bot messages, and system events get auto-cleared in these channels. Direct @-mentions to you stay unread.

Prefix matches the channel name (e.g. vsl- matches vsl-client-a).
Section name must match a section already in your Slack sidebar — exact spelling.
Auto-read does not affect direct mentions — Slack still notifies you on those.
